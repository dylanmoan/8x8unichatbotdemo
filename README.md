# 8x8 Uni Chatbot Demo

A simple, elegant website hosting an 8x8 AI Studio chatbot widget for instant customer support and engagement.

## Overview

This project provides a clean, responsive interface to embed and showcase your 8x8 AI Studio chatbot. Built with HTML and CSS, it's optimized for GitHub Pages hosting.

## Features

- 🤖 8x8 AI Studio chatbot widget integration
- 📱 Fully responsive design (mobile, tablet, desktop)
- 🎨 Modern, gradient-based branding
- ⚡ Fast loading and performance optimized
- 🌐 GitHub Pages ready deployment

## Getting Started

### Prerequisites

- A GitHub account
- Your 8x8 AI Studio widget ID

### Installation

1. Clone the repository:
```bash
git clone https://github.com/dylanmoan/8x8unichatbotdemo.git
cd 8x8unichatbotdemo
```

2. The website is ready to use! No additional dependencies or build steps required.

## Deployment

This site is configured for GitHub Pages deployment:

1. Go to your repository settings
2. Navigate to **Pages** section
3. Set **Source** to "Deploy from a branch"
4. Select **main** branch and **/(root)** folder
5. Click **Save**

Your site will be live at: `https://dylanmoan.github.io/8x8unichatbotdemo/`

## Customization

### Changing Colors

Edit the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #1e40af;      /* Main color */
    --secondary-color: #0ea5e9;    /* Secondary color */
    --accent-color: #06b6d4;       /* Accent color */
    --background-color: #f8fafc;   /* Background */
    --text-color: #1e293b;         /* Text color */
}
```

### Updating Content

Edit `index.html` to customize:
- Header title and subtitle
- Welcome message
- Footer text
- Widget ID (if needed)

### Styling

All CSS is in `styles.css`. The design includes:
- Gradient backgrounds
- Responsive breakpoints for mobile, tablet, and desktop
- Smooth animations
- Professional spacing and typography

## Troubleshooting

### Widget not appearing?
- Verify the widget ID is correct: `wgt_birch_a07ad9b7`
- Check browser console for errors
- Ensure JavaScript is enabled

### GitHub Pages not working?
- Verify repository is public
- Check Pages settings in repository settings
- Wait 1-2 minutes for changes to deploy
